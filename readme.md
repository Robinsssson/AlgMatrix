## 优化算法

优化算法的目标是解决一系列如
$$
    find\ \ \ \ \ \ \ \ X \in \ \R^3 \\
    s.t. \ \ \ \ \ \ \ \ \ \ G(X)<0 \\
    max/min \ \ \ \  fitness(X)
$$
